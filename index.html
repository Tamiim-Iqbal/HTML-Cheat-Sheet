🌐 HTML Cheat Sheet

Everything you need as a beginner web developer.

Basic Structure

<!DOCTYPE html>
<html lang="en" >
    <head>
        <meta charset="UTF-8">
        <title>Page</title>
    </head>
    <body>
        <!-- content here -->
    </body>
</html>>

Headings

<h1>
<h2>
<h3>
<h4>
<h5>
<h6>

There are six levels of headings. <h1> is the most important (largest), and <h6> is the least important (smallest).

Section Divisions

Attribute Purpose
<div> Division or Section of Page Content
<span> Section of text within other content
<br> Line Break
<hr> Basic Horizontal Line

Image

Attribute Purpose
<img> Inserts an Image
src Image location
alt Alternative text
<img src="URL" alt="" width="" height="">

Text Formatting

Attribute Purpose
<p> Defines a Paragraph
<strong> Defines Important Text (Semantic)
<em> Defines Emphasized Text (Semantic)
<small> Smaller Text
<u> Underlined Text
<mark> Highlighted Text

Lists

Attribute Purpose
<ul> Defines an Unordered List
<ol> Defines an Ordered List
<li> Defines a List Item

UL Types :

  • disc
  • circle
  • square

OL Types :

  1. 1
  1. a
  1. i

Link

Attribute Purpose
<a> Creates a Hyperlink
href The URL that the hyperlink points to
target Link target location: _self, _blank, _top, _parent
<a href="URL" target="_blank"> click me </a>

Semantic HTML

Attribute Purpose
<header> Masthead or important information
<nav> A section of navigation links
<main> The main content of the page
<section> Creates a section
<aside> Creates a sidebar
<footer> Creates a footer section

Media (Audio & Video)

Attribute Purpose
<audio> Creates an Audio Player
<video> Creates a Video Player
controls Displays audio/video controls
src The URL of the audio/video file
<audio controls src="URL"> </audio>
<video controls src="URL"> </video>

Input

Attribute Purpose
<input> Creates an Input Field
type Specifies the type of input field
placeholder Displays hint text inside an input field until the user types
<input type="text" placeholder="Enter your name" name="" id="">

Form

Attribute Purpose
<form> Creates a Form
<label> Creates a label for an input field
<fieldset> Groups related form elements
<form>
<label for="mail">Email: </label>
<input type="email" >
<label for="pw">Password: </label>
<input type="password">
<input type="submit" value="Login">
